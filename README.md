# AI Study Assistant

An AI-powered web app that helps students learn faster by generating explanations, notes, and quiz questions for any topic.

## Features
- Simple explanation of any topic
- Bullet-point notes
- Auto-generated MCQs with answers
- Fast and interactive UI using Streamlit

## Tech Stack
- Python
- Streamlit
- OpenAI API

## How to Run
1. Open a terminal in this project folder.
2. Install dependencies:
   - pip install streamlit openai
3. Set your API key:
   - Windows cmd: set OPENAI_API_KEY=your_real_key_here
   - Windows PowerShell: $env:OPENAI_API_KEY="your_real_key_here"
4. Start the app:
   - streamlit run app.py

## Example
Input: PCA
Output: Explanation + Notes + Quiz

## Use Case
Helps students quickly understand concepts and self-test knowledge.
